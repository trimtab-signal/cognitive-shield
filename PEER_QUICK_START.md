1. **Navigate to Heartbeat Tab**

2. **Get Your Connection Code:**
   - Your Peer ID is displayed at the top
   - Click "Copy Connection Code"
   - Share this with your host (via secure channel)

3. **Add Your Host:**
   - Click "Add Peer"
   - Paste the connection code your host provided
   - Enter their display name
   - Connection establishes automatically

4. **Set Your Status:**
   - Select your current status (Green/Yellow/Orange/Red)
   - Set check-in interval (default: 1 hour)
   - Your status will broadcast to connected peers

## üõ†Ô∏è Troubleshooting

### "Ollama Connection Failed"
- Ensure Ollama is running: `ollama serve`
- Check endpoint in Settings matches your Ollama URL
- Verify firewall allows localhost connections

### "Peer Connection Failed"
- Check WebRTC is enabled in your firewall
- Verify you have the correct connection code
- Try disconnecting and reconnecting

### "Status Percentage Not Calculating"
- Complete the Daily Check-In (all 15 questions)
- Check that responses are within valid ranges
- Verify œÄ-Metric calculation in browser console (if technical)

### "Messages Not Processing"
- Check LLM provider is configured correctly
- Verify API key (if using cloud LLM)
- Check browser console for errors
